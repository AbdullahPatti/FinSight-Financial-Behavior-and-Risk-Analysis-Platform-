# 💰 FinSight  
### Financial Behavior and Risk Analysis Platform

---

## 📌 Overview
**FinSight** is a data-driven financial intelligence platform that analyzes structured corporate financial data to uncover behavioral patterns, detect anomalies, and assess financial risk over time.

Instead of just reporting numbers, FinSight answers critical questions:
- Is the company financially healthy?
- Where is money being spent excessively?
- Are there hidden risks or anomalies in transactions?
- How is financial behavior evolving over time?

The system produces a **quarterly risk profile** classified into:
> **Low | Medium | High | Extreme**

---

## 🎯 Objective
To transform raw financial data into actionable behavioral insights using data analytics, machine learning, and statistical modeling.

---

## ❗ Problem Statement
Organizations store large volumes of financial data but fail to extract meaningful insights.

Traditional systems:
- Show what happened
- Do NOT explain whether it is healthy or risky

FinSight bridges this gap by:
- Automating financial health assessment  
- Detecting abnormal spending behavior  
- Identifying risk trends before they become critical  

---

## ⚙️ System Architecture
Data Ingestion → Preprocessing → Feature Engineering →  
Numerical Pipeline + NLP Pipeline → Risk Scoring → Dashboard

---

## 📊 Dataset
- ~50,000 transactions  
- 19 features  
- FY2021 – FY2024  
- 1,500 labeled anomalies
